DATABASE ARCHITECTURE:

Storage: SQLite (read_rates.db, ~50 MB)

Partitioned by: ACL_INSERT_DATE (when data inserted)

Sync Strategy:
  1. Query: SELECT MAX(acl_insert_date) → Get latest date
  2. Filter: WHERE acl_insert_date > latest_date
  3. Fetch: Pull new rows from BigQuery
  4. Append: Insert into SQLite (duplicates skipped)

Example:
  Local DB:  2026-01-01 to 2026-06-15
  GCS table: 2026-01-01 to 2026-06-19
  Sync: Fetches only new 4 days
  Result: ~27,360 new rows appended

GOOGLE CLOUD SYNC:

Manual Sync (UI):
  1. Go to: http://localhost:8000/admin/debug
  2. Enter GCS credentials
  3. Click "Sync Data"
  4. New rows automatically appended

Python API:

  result = sync_from_google_cloud()
  # Returns: {'status': 'success', 'inserted': 27360, ...}
